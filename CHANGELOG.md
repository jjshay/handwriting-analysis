## [1.0.0] - 2025-01-11

### Added

- Forensic-grade signature and handwriting feature extraction
- Statistical baseline generation from known authentic samples
- Feature analysis including slant angle, height ratios, stroke characteristics, and pressure patterns
- Z-score and Mahalanobis distance calculations for statistical comparison
- Probability-based confidence scoring for authentication opinions
- Multi-AI review system using Claude, GPT-4, Gemini, and Grok for consensus analysis
- Signature-specific analysis module for focused signature authentication
- Statistical signature generator for testing and research purposes
- Detailed forensic report generation with feature-by-feature comparison
- Support for questioned document analysis against established baselines
- Sample authentic signature images and baseline data for demonstration
- Demo mode for running without external dependencies
- Visual showcase script for quick demonstration
- Comprehensive test suite
- CI workflow for automated testing
